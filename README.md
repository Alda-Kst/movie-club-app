# 🎬 Movie Club App

A sophisticated movie discovery application built with **React**, featuring real-time data fetching from the [TMDB API](https://www.themoviedb.org/).

## 🚀 Live Demo

🔗 [Visit the Movie Club App](https://movie-club-app-34kg.vercel.app)

## ✨ Features

- **Real-time Search:** Instantly find movies by title.
- **Cinematic UI:** High-fidelity interface with responsive design.
- **Dynamic Loading:** Smooth loading states with spinners for a better user experience.
- **API Integration:** Seamlessly fetching the latest movie data and posters.

## 🛠️ Tech Stack

- **Frontend:** React, JavaScript (ES6+)
- **Styling:** CSS3 (Custom responsive layouts)
- **Data:** TMDB API (The Movie Database)
- **Deployment:** Vercel

## ⚙️ Installation & Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` file and add your `REACT_APP_TMDB_KEY`
4. Run the app: `npm start`
